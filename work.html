    Neuron

    Call Tracking and Analytics

    Neuron is a phone call tracking and analytics service.

    Neuron is powered by Twilio, Sails.js (a MVC web framework built on express), and Node.js. Sails.js allowed us to produce a REST-ful internal API with almost no effort. It also sped up development dramatically by allowing us to fully embrace Model-View-Controller.

    Wunder

    Its pretty wonderful

    Wunder is an educational webapp that focuses on practice based learning.

    This was a joint undertaking between 2 of Wunder Labs very own designers. Due to its interactive nature, we decided to build the backend with Node.js and the Express module to aid development. This enabled us to share Javascript code between the frontend and backend, and scale the website when we experienced traffic fluctuation.

    iiTutoring

    iiTutoring was a tutoring company based in Orange County. The website managed scheduling and hiring of tutors and had a live chat feature implemented via socket.io. iiTutoring was fully responsive and was built with many HTML5 and CSS3 features.
